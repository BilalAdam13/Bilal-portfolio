Live demo: Weather card web component

01

A small vanilla-JavaScript custom element (<weather-card>) that fetches current conditions for San Diego from the Open-Meteo API, parses the JSON, and renders it with proper loading and error states. No framework, no API key, no build step — just standards.

API: Open-Meteo (free, no key). The component handles the fetch, parses the JSON, maps WMO weather codes to human descriptions, and degrades gracefully on network errors.

Automated Toxic Comment Detector

  • Python
  • BERT
  • BiLSTM
  • NLP

Multi-label classifier for six toxicity categories trained on the Jigsaw dataset. Compared classical (TF-IDF + LogReg, SVM), deep (CNN, BiLSTM with attention), and transformer (BERT) approaches, evaluated with ROC-AUC to handle severe class imbalance. Improved rare-class performance through attention, class-balanced training, and hyperparameter tuning.
